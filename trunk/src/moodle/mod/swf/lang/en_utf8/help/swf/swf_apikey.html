SWF API Key

Function: API keys allow a SWF application to access web services from a provider over an Internet connection. Some service providers require an API key while others do not. An API key is a long unique string of text characters that identifies your web site domain name to the web service provider. It is similar, in a way, to a password. A typical example is the Google Maps API.

Parameter: Copy and paste the provided key

Default: none

Also see: To acquire a Google Maps API key - http://code.google.com/apis/maps/signup.html
