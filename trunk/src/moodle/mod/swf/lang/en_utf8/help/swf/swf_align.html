SWF Align

Function: Defines the text alignment around the SWF application.

Parameters: middle, left, right, top, bottom

Default: middle

Please note: This parameter affects bodies of text and/or items on the web page where the SWF application is embedded. For example, setting align to "left" will position the embedded SWF file to the left of the "Moodle Docs for this page", "You are logged in as..." and "Course" navigation links.
