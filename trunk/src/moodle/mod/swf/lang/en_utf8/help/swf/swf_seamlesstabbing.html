SWF Use Seamless Tabbing

Function: An accessibility parameter that allows users to tab out of the SWF application window. If set to false, the user will be unable to access browser controls via keyboard access (tabbing).

Parameters: true/false

Default: true

Please note: This setting must be set to true for Section 508 (U.S. 1973 Workforce Rehabilitation Act) compliance.
