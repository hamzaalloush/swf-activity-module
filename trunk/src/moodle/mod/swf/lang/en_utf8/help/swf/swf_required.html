SWF Parameters

Function: Defines the Flash application to be deployed, the dimensions of the Flash Player window and the minimum required Flash Player version.

Parameters:

  • SWF File: URL to main Flash learning application (opens file-picker to course files directory).
  • Width: width of Flash Player window in pixels or %.
  • Height: height of Flash Player window in pixels.
  • Full Browser: Flash Player fills 100% width and 100% height of browser window (Not to be confused with full screen)*
  • Version: Minimum required Flash Player version.**

Please note:

* If set to 100%, this renders full browser Flash with no Moodle navigation controls.

** If user's browser has a lower version, they will be prompted to download and install the latest version. This requires administrator rights.
