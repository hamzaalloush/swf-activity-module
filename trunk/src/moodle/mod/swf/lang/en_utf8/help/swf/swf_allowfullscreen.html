SWF Allow Full Screen

Function: Determines whether the SWF application can enter full screen mode, whereby it takes over the entire user's screen. It's the same function used on YouTube.com's video player.

Parameters: true, false

Default: false

Please note: This parameter only gives permission to enter full screen mode. The SWF application must have the necessary function included in it. Also, all keyboard events are suspended during full screen mode except the escape key (exit full screen mode).
