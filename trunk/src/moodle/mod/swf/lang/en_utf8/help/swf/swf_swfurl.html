SWF URL

Function: Defines the Flash or Flex Framework application to be deployed as a learning interaction.

Parameters: link to a SWF file

Default: none

Please note: The file picker tool only links to the course files directory. You should upload all SWF files for the course here, to a sub-directory named "swf/", i.e. swf/your_flash_file.swf
