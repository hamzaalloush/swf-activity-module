SWF Grading

Function: Passes a number from 0 - 100 to determine the grade weighting of a particular instance of the SWF Activity Module.

Parameters: numbers from 0 - 100

Default: 100

Please note: You probably don't want every SWF Activity to have the same maximum score. Consider this:

Activity #1 has 10 questions - set maximum grading to 50% or 10 points

Activity #2 has 20 questions - set maximum grading to 100% or 20 points

You can organise the grade weighting anyway you like for each activity in each course.
