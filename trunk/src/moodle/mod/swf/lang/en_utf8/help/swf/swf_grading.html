SWF Grading

Grade Type: Specifies the type of grade used:

  • none (no grading possible)
  • value (enables the maximum and minimum grade settings)
  • scale (enables the scale setting)
  • text (feedback only).

Only value and scale grade types may be aggregated. The grade type for an activity-based grade item is set on the update activity page

Scale: When using the scale grade type, a scale can be selected. The scale for an activity-based grade item is selected on the update activity page.

Maximum Grade: When using the value grade type, a maximum grade can be set. The maximum grade for an activity-based grade item is set on the update activity page.

Minimum Grade: When using the value grade type, a minimum grade can be set.

Grade to Pass: If an item has a grade that users must equal or exceed to pass that item, you can set that here.

Grade Display Type: Specifies how to display grades in the grader and user reports. Grades may be shown as actual grades, as percentages (in reference to the minimum and maximum grades) or as letters.

Decimal Points: Specifies the number of decimal points to display for each grade. This setting has no effect on grade calculations, which are made with an accuracy of 5 decimal places.

Feedback Text: Text to show user as feedback at any time during a Flash e-learning interaction.

Feedback Link: A link to a file or web page for feedback, e.g. help files, reference material, further study, etc.
