SWF Stage Align

Function: Defines the point where the display area of the embedded SWF application is anchored. If the Flash Player window is resized, then the display area will move according to the point defined.

Parameters: tl, tr, bl, br, l, t, r, b

  • tl - TOP_LEFT
  • tr - TOP_RIGHT
  • bl - BOTTOM_LEFT
  • br - BOTTOM_RIGHT
  • l - LEFT
  • t - TOP
  • r - RIGHT
  • b - BOTTOM

Default: tl
