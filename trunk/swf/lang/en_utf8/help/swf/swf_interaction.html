SWF Interaction (Not yet implemented)

Function: Defines the desired learning interaction data in the SWF activity module database. The data is usually texts, vocabulary, questions, answers and links to any necessary audio, images, video, etc.

Parameters: Selection list of available learning interactions (ID number)

Default: none

Please note: Not all interactions are compatible with all SWF applications. You must select an interaction from the list that is compatible with the selected SWF application. Alternatively, learning interaction data may also be provided in the form of an XML file.
