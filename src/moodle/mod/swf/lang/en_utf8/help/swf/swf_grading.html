SWF Grading

Grade Type: Specifies the type of grade used:

  • none (no grading possible)
  • value (enables the maximum and minimum grade settings)
  • text (feedback only).

Only value and scale grade types may be aggregated. The grade type for an activity-based grade item is set on the update activity page

Maximum Grade: When using the value grade type, a maximum grade can be set. The maximum grade for an activity-based grade item is set on the update activity page.

Minimum Grade: When using the value grade type, a minimum grade can be set.

Pass Grade: If an item has a grade that users must equal or exceed to pass that item, you can set that in the grade book:

  1. Administration > Grades > Grader Report > Turn editing on.
  2. Click on edit icon immediately under the SWF Activity Module instance (grade item) title
  3. Edit the grade item as required > Save

Conditional Sequencing: Adds this instance to a list of instances that must be completed in order, i.e. you can't attempt module 2 until you've completed module 1.

Feedback Text: Text to show user as feedback at any time during a Flash e-learning interaction.

Feedback Link: A link to a file or web page for feedback, e.g. help files, reference material, further study, etc.
